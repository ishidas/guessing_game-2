How I did it using JavaScript

  • By first defining the questions and total correct value as variables, the computer remembers and stores the answers.
  • The questions are presented in a prompt tag, followed by 'if' 'else' statements containing alert tags and point value assignment.
  • By using 'if' 'else' statements, the computer can determine which answers are correct and which are incorrect.
  • The correct answers are valued with a 1 and incorrect answers are valued with a 0 in the 'if' 'else' statements.
  • The variable 'score' is taking the sum of the questions (1 for correct, 0 for incorrect), and alerting the user about their score.
